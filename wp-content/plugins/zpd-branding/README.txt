Brands the client dashboard

== Description ==

This plugin is intended for Zero Point Development customers only.

It customises the login logo, removes default WordPress dashboard widgets and the WordPress admin menu logo.

It adds a WP Wingman News dashboard widget which gets the RSS feed from the Lime Canvas website.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `zpd-branding.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
